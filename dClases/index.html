Explicación

Clases y sus significados

Nombre clase Significado
Cliente Almacena a los clientes o usuarios del sistema que realicen tramites al area de urbanismo de la alcaldia de Masaya.
TelefonoCliente Representa los telefonos que puede tener un Cliente.
Tramite Generalización de los tipos de tramites que se cubren en el analisis del sistema.
TramiteCliente Une un cliente con un tipo de tramite, es decir, el tramite solicitado por el cliente.
TramiteVigencia Aqui se asocian los distintos periodos de vigencia con la clase padre de tramite.
PeriodoVigencia Si es un tipo de tramite que necesita establecer un periodo de tiempo en el que estara vigente ese tramite.
Denuncia Tipo de tramite para realizar una denuncia de una inconformidad. Se deben ingresar los parametros para poder localizar el lugar de la denuncia.
RoturaCalle Tipo de tramite para roturas de calles, se debe especificar el area de la rotura.
MatriculaLocal Sirve para constatar que se tiene un permiso de tener un local.
Proyecto Se realiza a partir de un TramiteCliente y representa el proyecto en marcha.
Encargado Datos sobre la persona encargada de un proyecto. Un proyecto solo puede tener un unico encargado.
