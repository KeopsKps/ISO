Significado de casos de usos

Se presenta una explicación de los casos de usos por parte de los actores y de un caso de uso en concreto.

Actor Caso de Uso Explicación
Usuario Ingresa al Sistema El usuario puede usar el sistema web para entrar al sistema y realizar los primeros pasos del tramite.
Usuario Resolución en linea Falta la explicación
Usuario PreSolicitud El usuario realiza una PreSolicitud del tramite que desea realizar. De esta forma se puede responder antes si el tramite el valido o no.
Usuario Solicitud El usuario realiza un Solicitud del tramite, ingresando los datos necesarios para continuar.
Usuario Resolución El usuario puede ver la resolucion de la solicitud realizada previamente.
Usuario Permiso El usuario recibe permiso una vez que se ha realizado la revisión.
